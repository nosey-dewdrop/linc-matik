# linc-matik

people are strange. i never see much things coming. this is an app that uses the claude api to roast me with sharp, challenging feedback - designed as a mental toughness trainer. the goal is to help you practice handling harsh criticism so you become more resilient in real life.

- schopenhauer / how to win an argument: people will use dirty tricks in arguments. learn the strategies and don't get caught off guard. always be prepared the worst thing coming. you must expect anything from anyone.
- nietzsche / be raw. most people aren't actually trying to destroy you, this is how they act in their point of views. for instance, if i am a gila monster, i will eat you. not because i want to hurt you. but it is what should be done for me. 
- prince myshkin / he is the best person. this was his crime.
